<h2>Wie du mitmachst</h2>

<ol>
  <li>Bevor du etwas postest:</li>
  <ul>
    <li>Schau, ob dein Thema schon existiert</li>
<li>👍 Like bestehende Beiträge statt Duplikate zu erstellen</li>
  </ul>
    <li>Zum posten beachten:</li>
  <ul>
    <li>🐞 Bug gefunden? → Erstelle ein Issue mit dem Bug-Template</li>
<li>💡 Coole Idee? → Teile sie mit dem Idee-Template!</li>
<li>💬 Andere Meinung? → Nutze die Discussions</li>
  </ul>
  <li>Regeln (kurz & schmerzlos)</li>
  <ul>
    <li>Bleib konstruktiv und respektvoll, wir sind alle hier um Spaß zu haben</li>
<li>Beschreibe Probleme so genau wie nötig und so knapp wie möglich</li>
<li>Memes sind erlaubt… solange sie hilfreich sind</li>
    <li>Beschränke dich auf wichtige, dem Feedback förderliche Nachrichten, kein chatten oder spammen</li>
  </ul>
</ol>

<h2>Was passiert mit deinem Feedback?</h2>
  <ul>
    <li>Deine Einsendung wird von uns geprüft und eingestuft – Wir lesen alles.</li>
    <li>Die Community bestimmt die Dringlichkeit mit Likes 👍</li>
    <li>Das wichtigste Feedback wird in den nächsten Zyronia PnP Produkten berücksichtigt</li>
  </ul>

Status-Labels zeigen dir:

geplant → kommt wahrscheinlich<br>
in arbeit → wir arbeiten dran<br>
erledigt → findet in der nächsten Auflage Anwendung<br>
intendiert → ist so gewollt (wir sagen dir wieso)
